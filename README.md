# Simple JDBC Project

This is a simple Java JDBC project that connects to a MySQL database, creates a table, inserts a record, and retrieves data.

---

## 🛠️ Technologies Used

- Java
- JDBC
- MySQL
- MySQL Connector/J
- SQL
- IntelliJ IDEA

---

## 🚀 Key Highlights

- Core Java and JDBC integration
- MySQL database connectivity
- Automatic table creation
- Data insertion into the database
- Data retrieval and display
- Console-based application
- Beginner-friendly project for learning JDBC
